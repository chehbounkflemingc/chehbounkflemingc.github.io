Practical Lab 1

Items completed in Week 1:

  1. Custom map creation with Google using Google MyMaps Tutorial
  2. Review of the three pillars of Google Maps as a platform: Maps, Routes and Places and creation of an introductory example for each of the following "getting started" items: JavaScript API (HTML Interactive Map) , Directions API (JSON Return), and Places API (JSON Return)

Items completed in Week 2:

  1. Addition of a simple Google map with a marker to a web page using the first tutorial.
  2. Using marker clusters to display a large number of markers on a map using the second tutorial.
  3. Applied three techniques to vizualise data on Google maps: basic markers, sized circles, and a heatmap.
  4. Creation of a story map using the Google Maps JavaScript API.
